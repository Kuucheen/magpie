# Frontend

Magpie's frontend is an Angular `21.1` application.

Current stack:

- Angular `21.1`
- PrimeNG `21`
- Tailwind CSS `4`
- npm-based workflow

## Prerequisites

- Node.js `20.19+` or `22.12+`
- npm

## Development server

```bash
npm install
npm run start
```

The dev server runs at `http://localhost:4200/`.

## Build

```bash
npm run build
```

The production browser build is emitted to `dist/frontend/browser/`.

## Tests

```bash
npm test
```
